black_scholes(S0, r, volatility, T, N, L, rand_gen_state=None)

source code 

Generates paths for S in risk-neutral Black-Scholes formulation: dS = r(t)*S(t)*dt + volatility*S(t)*dW_t

Generates N/2 paths plus their antithetic paths, so a total of N paths are returned. If N is odd, an error is raised. Note that the time steps are numbered j=0,...,L, giving L+1 time steps in total.

Parameters:
  • S0 (number) - the starting value of the generated paths,
  • r (number) - the risk-free interest rate,
  • volatility (number) - the volatility of the underlying,
  • T (number) - the maturity time, measured in years,
  • N (integer) - the number of paths to generate,
  • L (integer) - the number of time steps,
  • rand_gen_state - NumPy random number generator state object.
Returns:
a tuple containing a (L+1) x N-array of paths, and the random number generator state used to generate the paths.

jump_diffusion(S0, r, volatility, d, eta, theta, T, N, L, rand_gen_state=None)

source code 

Generates paths for S in a risk-neutral jump-diffusion with non-negative, exponentially distributed jumps, and continuous dividend payment.

S_t = exp(volatility*W_t + mu*t + J_t) where W is a Wiener process, J a compound Poisson process with jump intensity eta > 0 and increments following exponential distribution with parameter theta > 1, and mu = r - d - volatility^2/2 + eta/(1-theta).

Parameters:
  • S0 (number) - the starting value of the generated paths,
  • r (number) - the risk-free interest rate,
  • volatility (number) - the volatility of the underlying,
  • d (number) - the continuous dividend payment rate of the underlying,
  • eta (number) - jump intensity of the underlying,
  • theta (number) - exponential distribution parameter of the jump sizes,
  • T (number) - the maturity time, measured in years,
  • N (integer) - the number of paths to generate,
  • L (integer) - the number of time steps,
  • rand_gen_state - NumPy random number generator state object.
Returns:
a tuple containing a (L+1) x N-array of paths, and the random number generator state used to generate the paths.
