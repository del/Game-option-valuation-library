 91          """ 
 92          Calculates the payoff M{R(sigma_j,tau_j)} from the GCC at time M{j} 
 93          using the stopping strategies in C{sigma} and C{tau}. 
 94           
 95          @type           X:                      (L+1) x N-array 
 96          @param          X:                      the payoffs to the option holder when the writer terminates, 
 97          @type           Y:                      (L+1) x N-array 
 98          @param          Y:                      the payoffs to the option holder when he exercises, 
 99          @type           sigma:          (L+1) x N-array 
100          @param          sigma:          the optimal stopping strategy for the writer of the option, 
101          @type           tau:            (L+1) x N-array 
102          @param          tau:            the optimal stopping strategy for the holder of the option, 
103          @type           j:                      integer 
104          @param          j:                      the time step to evaluate the payoff at. 
105           
106          @return:        an N-array containing the payoffs. 
107          """ 
108          N = X.shape[1] 
109          I_sigma_lt_tau = np.where(np.less(sigma[j, :], tau[j, :]), 1, 0) 
110          I_tau_lte_sigma = 1 - I_sigma_lt_tau 
111          R_sigma_tau_j = X[sigma[j, :], range(N)]*I_sigma_lt_tau + Y[tau[j, :], range(N)]*I_tau_lte_sigma 
112          return R_sigma_tau_j
