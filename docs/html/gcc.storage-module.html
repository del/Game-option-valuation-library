csv_to_file(dict, csv_file, headers=False)

source code 

Saves a Python dictionary as CSV in a file.

Parameters:
  • dict (dict) - the dictionary,
  • csv_file (string) - the path where the CSV file will be saved,
  • headers (boolean) - whether or not to start a new CSV file and write the keys of the dictionary as headers in it.
Returns:
nothing
Notes:
  • If the headers parameter is true, the file will be truncated, and the dictionary keys will be written on the first row, as headers for the CSV, with the values on the second row.
  • If the headers parameter is false, the values of the dictionary will be appended on a new row at the end of the file.
  • The dictionary will be written according to the sorted list of keys, to ensure that several dictionaries with the same keys can be written without scrambling of values.
  • This function does not verify that the dictionary being written is a fit for previously written values in the file.
