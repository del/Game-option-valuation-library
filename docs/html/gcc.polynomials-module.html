lse(S_t, Y_tau, m, poly_type)

source code 

Calculate LSE a in R^m for problem Y_tau = S_t*a + err, where Y_tau is the stopped payoff at stopping time tau^{bar}_{t+1} for all paths omega_n, n=1, ..., N and S_t is the stock price for all paths.

Parameters:
  • S_t (N-array) - the stock price at time t for all paths,
  • Y_tau (N-array) - the stopped payoff at stopping time tau,
  • m (integer) - the number of polynomials to evaluate,
  • poly_type (string) - the type of functions in the projection subspace, as recognised by gcc.polynomials; e.g. "hermite" or "laguerre".
Returns:
an array where the first element is the N-array LSE a and the second element is the N-array of the projection Y_fit = S_t*a.
