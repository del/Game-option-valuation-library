value(S, K, delta, r, T, **params)

source code 

Values a callable put option.

Parameters:
  • S ((L+1) x N-array) - the simulated underlying paths. L is the number of time steps - 1 (timesteps are numbered 0, 1, ..., L), and N is the number of simulated paths,
  • K (number) - the strike of the put component,
  • delta (number) - the penalty for calling the option,
  • r (number) - the risk-free interest rate,
  • T (number) - the maturity time, measured in years,
  • params (dict) - optional parameters to pass on to the output.
Returns:
the output of gcc.valuation.value_gcc
